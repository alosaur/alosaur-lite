# Alosaur-lite

Lightweight version of Alosaur. Specially designed to work with deno deploy.

Supports:
- [x] Controllers
- [x] Actions methods (GET, POST, etc)
- [ ] Serve static files
- [ ] Other decorators  
- [ ] Render pages
- [ ] Middlewares:
- - [ ] CORS
- - [ ] Static files
- - [ ] Custom middleware
